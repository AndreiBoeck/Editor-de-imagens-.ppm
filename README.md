# Manipulação de imagens em c
Este projeto implementa diversas funções para manipulação de imagens ppm em C. Ele permite que o usuário aplique diferentes efeitos a uma imagem, como conversão para tons de cinza, geração de negativo, ajuste de brilho, rotação e envelhecimento da imagem. 

Este trabalho é destinado à cadeira de programação em baixo nível.

## Como Usar
Compile o código utilizando um compilador C e execute o programa. O programa solicitará o nome do arquivo de imagem PPM a ser processado e apresentará um menu com as opções de manipulação. 

Escolha a operação desejada, a imagem processada será salva em um novo arquivo chamado ImgAlterada.ppm. O programa pode ser rodado quantas vezes for desejado, sempre fazendo alterações a partir da imagem original.
